[WIP] Tech Insight #1 - RAM

"Tahun 2023, minimal RAM 8 GB lah", "DDR5 mahal bos", "Lemot?? upgrade RAM lah", "Windows 11 sekarang banyak makan RAM", adalah beberapa quote yang saya temukan pada group sosial komputer.

Apa itu RAM?

Random Access Memory (RAM) merupakan komponen yang wajib ada untuk dapat menjalankan komputer. Menurut techtarget.com, RAM adalah perangkat keras komputer dimana Operating System (OS), aplikasi, dan data yang sedang digunakan disimpan sehingga prosesor dapat mengakses data tersebut dengan sangat cepat.

RAM merupakan komponen yang bersifat volatile. Volatile ini berarti RAM hanya dapat menyimpan data ketika diberikan arus listrik. Ketika arus listrik ini hilang, maka data yang disimpan juga ikut hilang. RAM menghapus datanya ketika komputer melakukan restart dan shutdown

Source: crucial

Kenapa RAM itu penting?

RAM merupakan salah satu komponen utama yang digunakan ketika komputer sedang melakukan booting. Booting dilakukan untuk menjalankan OS yang sudah di-install. Proses ini akan mengambil data dari penyimpanan OS dan dipindahkan ke RAM [2].

Storage OS -> RAM

Hal ini dilakukan untuk mempercepat akses data ketika diproses oleh prosesor. Hal ini dilakukan supaya prosesor dapat mengakses data OS lebih cepat. Kecepatan RAM jauh lebih cepat dibandingkan storage OS pada umumnya

Seberapa cepat RAM itu?

Berdasarkan crucial.com, transfer rate RAM DDR4 2400MHz adalah 19,2 GBps. Sedangkan pada SSD terbaru seperti Samsung 990 PRO yang sudah menggunakan PCIe 4.0, memiliki transfer rate sebesar 7,45 GBps.

Source: crucial

Apa itu DDR4, DDR5 yang sering disebut ketika membeli RAM?

DDR4 merupakan kepanjangan dari Double Data Rate 4. Angka 4 dibelakang merujuk kepada generasi dari DDR, sehingga dapat disebut sebagai generasi ke-4 dari DDR. Sehingga angka 5 pada DDR5, adalah generasi ke-5 dari DDR.

Karena perbedaan pada generasi, bentuk fisik pada RAM tersebut juga berbeda. Sehingga motherboard dengan konfigurasi DDR4 tidak bisa dipasang RAM DDR5.

Source: teamgroupinc.com

Apakah RAM Laptop dan Desktop berbeda?

RAM pada laptop dan desktop berbeda. Desktop menggunakan UDIMM, sedangkan laptop menggunakan SODIMM. UDIMM adalah singkatan dari Unbuffered Dual In-line Memory Module. SODIMM adalah singkatan dari Small Outline Dual In-Line Memory Module. Singkatnya, SODIMM merupakan bentuk kecil dari UDIMM [3].

Bagaimana cara kerja RAM?

Pada saat tampilan desktop OS muncul, RAM sudah diisi dengan data-data penting OS yang akan terus digunakan. Windows 11 pada saat ini membutuhkan minimal 4 GB RAM. Tidak hanya OS, RAM juga akan diisi oleh aplikasi yang akan digunakan seperti Notepad, Paint, Microsoft Office, dan games.

Pada saat sedang membuka aplikasi, ada delay sebelum aplikasi tampil di layar. Terdapat proses memasukkan data aplikasi ke dalam RAM untuk mempercepat performa dari aplikasi. Apabila aplikasi tidak menggunakan RAM, maka akan terjadi penurunan performa secara drastis. Hal ini dapat terjadi pada komputer yang memiliki RAM kecil/penuh.

Apa yang terjadi jika RAM penuh?

Virtual Memory atau page file, merupakan sebuah RAM virtual yang menggunakan ruang dari penyimpanan (bisa HDD atau SSD). Virtual Memory ini dikelola oleh OS. Jika penggunaan RAM sudah penuh, OS akan membuatkan Virtual Memory pada HDD atau SSD yang ada di komputer [4].

OS akan mengarahkan aplikasi-aplikasi yang membutuhkan RAM untuk menggunakan Virtual Memory. Hal ini merupakan kasus yang sering ditemukan pada kendala komputer lemot (khususnya yang menggunakan HDD).

Lemot ini diakibatkan oleh HDD yang memiliki performa jauh lebih pelan dibandingkan RAM. OS dan program saling berebutan untuk mendapatkan Virtual Memory yang membuat OS menjadi tidak responsive / hang.
