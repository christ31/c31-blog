Satu Tahun Bersama Linux

Tahun 2022, adalah tahun dimana saya memutuskan untuk menggunakan Linux sebagai OS utama. Mulai dari OS Linux yang populer untuk pemula, Ubuntu, hingga Fedora yang sekarang saya gunakan.

Jadi, apa itu Linux?

Menurut Linux.com, Linux adalah sebuah Operating System (OS) sama seperti Windows dan MacOS. Dari sumber yang sama, OS merupakan sebuah jembatan antara software dan hardware. Tanpa OS, software tidak akan bisa bekerja.

Kenapa beralih ke Linux?

Sebenarnya, keputusan saya untuk beralih ke Linux sepenuhnya diawali dari peluncuran Windows 11 pada tanggal 5 Oktober 2021. Pada saat itu, windows 11 cukup populer karena minimal requirement yang dibutuhkan cukup tinggi. Mulai dari wajib adanya perangkat TPM, hingga generasi prosesor Intel minimal sudah generasi 8.

Sebagai pengguna laptop dengan prosesor Intel generasi 7, tentu saja laptop saya tidak didukung oleh Windows 11. Hal inilah yang membuat saya beralih untuk menggunakan Linux, Ubuntu pada saat itu.

Windows 11 merupakan penggerak saya dalam menggunakan Linux sebagai OS utama
Sumber: blogs.windows.com

Kenapa Ubuntu?

Saya mengenal Ubuntu pada saat saya masih kecil, sekitar tahun 2005-2008. Di rentang waktu itu juga, saya juga dikenalkan dengan Fedora. Ubuntu dan Fedora yang saya gunakan pada saat itu sudah menggunakan GNOME sebagai Desktop Environtment (DE) nya.

Sepanjang sekolah, saya selalu bereksperimen dengan Linux. Saya mulai mencoba berbagai macam linux yang lainnya seperti Mandriva, OpenSuse, PCLinuxOS, dst. Pada akhirnya, saya selalu kembali ke Ubuntu karena banyaknya panduan untuk Ubuntu di Internet ketika ada masalah.

Sudah 1 tahun pakai Linux sebagai OS utama, gimana pengalamannya?

Lebih tepatnya sudah 1 tahun 9 bulan, berawal dari Ubuntu (Oktober 2021 - Mei 2022), kemudian ganti ke Fedora (Mei 2022, Juli 2023). Keduanya berasal dari dua induk yang berbeda. Fedora adalah cabang dari Red Hat, sedangkan Ubuntu adalah cabang dari Debian.

Selama menggunakan Linux, ada banyak sekali tantangan yang saya hadapi. Mulai dari mic headset yang tidak bisa digunakan, Wi-fi disconnect sendiri, Game dengan Hybrid GPU (Nvidia + Intel), Black Screen, dst. Problem ini beserta solusinya saya dokumentasikan pada repository GitHub saya.

Untuk penggunaan sehari-hari, hampir semua kebutuhan saya sudah dapat terpenuhi mulai dari Browsing, Gaming (via Wine atau Proton), Office (LibreOffice / Google Office Suite), Video Editing (Kdenlive), Image Editing (Krita / GIMP), Recording (OBS / Audacity), dan Programming (VS Code).

Beberapa kebutuhan yang kurang terpenuhi terletak pada Game yang membutuhkan performa komputer tinggi, atau memang belum didukung di Linux. Selain game, kebutuhan menggunakan Microsoft Office juga tidak dapat menggunakan Linux (Walaupun bisa menggunakan Windows Virtualization).

Di dalam 1 tahun ini, Linux memubat saya belajar banyak hal dan melakukan eksplorasi baik pada alternatif software maupun untuk troubleshooting ketika ada masalah. Jika sebelumnya hanya menggunakan Microsoft Office, di Linux saya belajar menggunakan LibreOffice. Jika sebelumnya desain tampilan window tidak dapat dikustomisasi selain light/dark theme, di Linux saya belajar CSS untuk membuat desain tampilan window sendiri.

Di Windows, sudah disediakan berbagai macam settings menggunakan Graphical User Interface (GUI). Di Linux, saya diharuskan untuk membiasakan diri menggunakan Command Line Interface (CLI). Sebuah alat yang harus digunakan dengan hati-hati.

Linux memberikan banyak sekali kebebasan dalam melakukan semuanya. Sudah ada banyak sekali variasi yang dapat dipilih oleh pemula. Walaupun begitu, penggunaan Linux ini sangat bergantung kepada penggunanya. Menurut saya, Linux memberikan pengalaman yang lebih baik dibandingkan Windows. Namun untuk pengguna lain, bisa jadi memilki pengalaman yang kurang baik. Semua itu ada di tangan penggunanya.
